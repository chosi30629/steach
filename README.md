# Steach(Student & Teacher)
### 비트캠프 최종 프로젝트
     구성원 : 김해동, 박한준, 안규영, 조성일
     개요 : 학교나 학원 등 교육기관에서 선생님이 해당 반의 수업 내용, 수업 자료, 조별 활동, 학생 관리를 편하게 할 수 있도록 
            도와주는 웹 애플리케이션입니다.
